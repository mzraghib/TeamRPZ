Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app\src\main\AndroidManifest.xml
* res\ => app\src\main\res\
* src\ => app\src\main\java\
* src\com\example\.idea\compiler.xml => app\src\main\resources\com\example\.idea\compiler.xml
* src\com\example\.idea\copyright\profiles_settings.xml => app\src\main\resources\com\example\.idea\copyright\profiles_settings.xml
* src\com\example\.idea\example.iml => app\src\main\resources\com\example\.idea\example.iml
* src\com\example\.idea\misc.xml => app\src\main\resources\com\example\.idea\misc.xml
* src\com\example\.idea\modules.xml => app\src\main\resources\com\example\.idea\modules.xml
* src\com\example\.idea\workspace.xml => app\src\main\resources\com\example\.idea\workspace.xml
